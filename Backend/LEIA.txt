Vou passar alguns pontos para o python funcionar:

1) Baixar o Python na versão 3.13.3 (ou a mais atualizada), quando instalar, permita
uma opção chamada PATH.

2) Após baixado o python em sua máquina, abra o VSCODE e aperte F1, devemos criar
um ambiente para baixarmos as bibliotecas necessárias, selecione o "CREATE ENVIROMENT"
e "VENV".

3) Abra o terminal, e baixe as seguintes bibliotecas:
* pip install -U langchain-google-genai
* pip install langchain google-generativeai
* pip install langchain-community
* pip install matplotlib
* pip install dotenv
* pip install pytrends

4) Rode o Main para testar, você pode ver o gráfico sendo gerado, basta retirar
o comentário ao final do código. A principal variável a ser transmitida deverá ser:
"result_product_final_trends".
